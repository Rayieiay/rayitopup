Tutorial Cara Top-up

1. Masukkan User ID dan Zone ID Anda, Contoh : 667632423 ( 5246 )

2. Pilih Nominal Diamonds yang kamu inginkan

3. Selesaikan pembayaran

4. Diamonds akan ditambahkan ke akun Mobile Legends kamu

INFORMASI PENTING!

1. Pastikan Level Akun Mobile Legends kamu sudah mencapai level 5

2. Kamu tidak bisa Top Up Weekly Diamond jika sudah 0/10 (70 Hari) kalian bisa check di bagian weekly diamond masing masing akun

Contoh : Jika masih 3/10 berarti kalian masih bisa top up 3x lagi

3. TOP UP ORIGINAL SERVER :

Hanya bisa untuk top up original server, tidak bisa untuk advance server

Lengkapi Data

Untuk mengetahui User ID Anda, silahkan klik menu profile di bagian kiri atas pada menu utama game. User ID akan terlihat di bagian bawah Nama karakter game Anda. Silahkan masukan User ID dan Server ID Anda untuk menyelesaikan transaksi. Contoh: 12345678(1234), maka ID = 12345678 dan Server = 1234
